CSV Generator for Teachers

Our Excel CSV export sample creates a CSV file that you can import as a class roster into a classroom management tool such as TeacherKit.

Select your classroom management tool


Create class roster

The sample creates a table for your chosen management tool. Click the button to create the table.


When you've filled in your roster, export a CSV file to your computer and then import into the tool.


Ready to export your roster?

Excel exports your class roster as a .CSV file. Click the help button to learn how.
